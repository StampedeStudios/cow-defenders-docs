Design Pillars

Classic tower defense game

Cow Defenders is a classic tower defense game. The main goal of the player is to defend his farm from enemy waves adopting different strategies.

Lost century vibes

Elements existing in the game are built with high-tech products that were once common but are now almost forgotten. Each turret makes use of reclaimed components and old materials.

80s display UI

UI is designed to remind old green displays. Noise and distortion effects are present in many parts of the in-game UI and menu UI.
