Accessibility

Some factors have been considered during the game development, a specific palette of color is being utilized so to make more clear what ships are considered hostile and what represents the Hero. Other aspects such as font readability and UI elements accessibility have been evaluated but the game can still be improved. A "How to play" section is also present in the main menu to help players learn the key bindings.

Tools

Following is a list of tools that can be used to make the game more accessible.

  • Adobe Color is a free website that helps pick specific color contrast and readability of shapes.
