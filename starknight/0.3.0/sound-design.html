Sound Design

A few sound effects are present to help the player empathize with the hero. The game lacks a main soundtrack.

Tools

Following is a list of tools that can be used to create sound effects for the game.

  • Audacity is a free and open-source audio editor. It is used to create and edit audio clips.

  • ChipTone is a free tool for generating sound effects.
